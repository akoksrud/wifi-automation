# Bring up the TIG stack

### Docker compose

While in the directory of the <kbd>docker-compose.yml</kbd>, run <kbd>docker compose up -d</kbd> to start the containers according to the instructions in the docker-compose file.

```bash
cd ~/tig-stack
docker compose up -d
```
