# Create SSH keypair

#### Prerequisites

You need some SSH libraries like OpenSSH installed to run these commands. They should be the same on Windows, Mac or Linux

#### Create a SSH keypair for your user

```bash
PS C:\> ssh-keygen -t ed25519
```
